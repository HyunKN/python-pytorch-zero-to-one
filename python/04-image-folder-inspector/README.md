# 04 — 이미지 폴더 검사

폴더를 순회하며 이미지 파일을 찾고 확장자별 개수를 요약합니다.

## 구현할 기능

- 하위 폴더까지 파일 탐색
- `.jpg`, `.jpeg`, `.png`, `.webp`만 선택
- 확장자를 소문자로 통일해 개수 집계
- 전체 이미지 수와 확장자별 개수 반환

## 배울 내용

- `pathlib.Path`
- `rglob()`
- 문자열 정규화
- 파일 filtering

## 완료 기준

- [ ] 대소문자가 다른 확장자를 같은 종류로 처리
- [ ] 이미지가 아닌 파일 제외
- [ ] 입력 폴더 내부를 수정하지 않음
