# 02 — List와 Dictionary 요약

상품 정보를 담은 `list[dict]`를 순회하며 필요한 정보를 집계합니다.

## 구현할 기능

- 전체 재고 수량 계산
- 가격이 가장 높은 상품 찾기
- 재고가 기준 이하인 상품 이름 수집
- 결과를 하나의 `dict`로 반환

## 배울 내용

- list와 dict 접근
- `for`, `if`
- accumulator와 비교
- 함수의 입력과 반환값

## 완료 기준

- [ ] 입력 list를 수정하지 않음
- [ ] 빈 list가 아닌 예제에서 정확한 집계 반환
- [ ] 각 반복문에서 갱신되는 값을 설명
