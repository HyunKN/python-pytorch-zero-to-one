# 08 — Tensor 기초

PyTorch Tensor를 만들고 shape, dtype, device와 기본 연산을 확인합니다.

## 구현할 기능

- Python list와 NumPy array를 Tensor로 변환
- shape, dtype, device 요약
- reshape와 matrix multiplication 수행
- CPU와 사용 가능한 accelerator 확인

## 배울 내용

- `torch.Tensor`
- shape와 dimension
- dtype와 device
- NumPy와 Tensor의 차이
